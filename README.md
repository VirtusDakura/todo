# To-Do List App

A simple and interactive to-do list application built with vanilla JavaScript, HTML, and CSS.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as completed
- ✅ Delete tasks
- ✅ Persistent storage using localStorage
- ✅ Keyboard support (press Enter to add tasks)
- ✅ Clean and responsive UI

## How to Use

1. Open `ToDoList.html` in your web browser
2. Type a task in the input box
3. Click the "Add" button or press Enter to add the task
4. Click on a task to mark it as complete (strikethrough)
5. Click the × button to delete a task
6. Your tasks are automatically saved and will persist even after closing the browser

## Files

- `ToDoList.html` - Main HTML structure
- `ToDoList.css` - Styling for the app
- `ToDoList.js` - JavaScript functionality and logic

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage API

## Browser Support

Works on all modern browsers that support localStorage and ES6 JavaScript.
